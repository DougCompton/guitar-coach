# Theory Tied to Playing

Use theory in a way that immediately touches the fretboard.

## Core principle

Never teach theory as isolated trivia when a fretboard example would help more. Tie every concept to one of these:

- intervals on one string
- triads on the top 3 strings
- chord construction from shapes the player already knows
- scale degrees inside a chord loop
- ear-first identification before explanation

## Default theory moves

### Intervals on one string
- Pick one string.
- Name the start note.
- Move to the target interval.
- Say the interval before playing it.
- Then play it in time.

### Triads on the top 3 strings
- Use strings 1 to 3, 2 to 4, or 3 to 5.
- Name the root, third, and fifth.
- Keep shapes tiny and movable.
- Connect the triad to the chord already used in the lesson or song.

### Chord construction from known shapes
- Start from an open or familiar movable chord.
- Explain which notes create major, minor, or dominant color.
- Remove or add one note and connect that change to what the ear hears.

## Coaching rule

End every theory block with a mini-win that proves the concept lived on the instrument, not only in words.
