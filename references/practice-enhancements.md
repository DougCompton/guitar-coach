# Practice Enhancements

Use this file to apply the upgraded coaching workflow.

## 1. Weak-spot memory

After each session, identify the biggest repeated problem from the log and carry it into the next session as a short corrective drill.

Priority weak spots:
- chord transitions
- rhythm and timing
- string noise or muted notes
- tension or posture
- picking control
- memory or confidence

Prefer one weak-spot drill, not many.

## 2. Daily start-here mode

When the user wants to begin immediately, run `scripts/start_here.py` first so the skill chooses the roadmap, lesson, weak spots, repertoire lane, warm-up, exact section times, and mini-wins automatically.

## 3. Tempo-based advancement

For rhythm-sensitive skills, require a tempo ladder instead of relying only on self-report.

Advance only when:
- the user hits the target BPM cleanly
- the reps are repeatable, not lucky
- timing stays steady through changes or accents
- tension stays low

If the user fails at a rung, drop one rung and rebuild.

## 4. Separate mastery scores

Score chords, rhythm, scales, fingerstyle, repertoire, and fretboard knowledge independently. Let the weakest active area influence the next session design.

## 5. Repertoire lane

Keep song work in three lanes:
- learning
- polishing
- maintenance

Use learning for the main application block, polishing for stabilization, and maintenance for easy wins or review days.

## 6. Theory tied to the instrument

Teach theory through intervals on one string, triads on the top 3 strings, and chord construction from shapes the user already knows. End the block with proof on the fretboard.

## 7. Spaced repetition

Once a lesson becomes playable, schedule quick reviews at day +2, +7, +14, and +30.

At each review:
- play the skill once at an easy tempo
- test it once at the current working tempo
- rate whether it stayed clean and confident

## 8. Weekly review

At least once every 7 days, summarize the recent logs and answer:
- what improved
- what keeps failing
- what should stay in active practice
- what can move into review mode
- whether the roadmap should change

## 9. Bad-day fallback

When motivation, time, or energy is low, protect the habit with a short session that avoids new material.
