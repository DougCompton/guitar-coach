# Daily Log Template

Use this reference when the user asks for the markdown file or wants the session written out as a daily record.

Filename pattern:

`YYYY-MM-DD-guitar-practice.md`

Minimum required fields:

- date
- active roadmap
- current lesson
- repertoire lane target
- practice goal
- planned minutes and actual minutes
- tags
- section-by-section notes
- mini-win result per section
- self-rating per section
- pain or tension check
- coach recommendation
- optional mastery-signal notes for chords, rhythm, scales, fingerstyle, repertoire, or fretboard knowledge
- optional overall rating (1-10) for trend charts
- optional readiness notes when the day started with a readiness check

Keep the markdown clean and easy to append to over time.

Required tag patterns:

- `#roadmap/<name>` such as `#roadmap/beginner`
- `#lesson/<slug>` such as `#lesson/chord-transitions`
- zero or more `#issue/<slug>` tags such as `#issue/rhythm`
- exactly one status tag such as `#status/repeat` or `#status/advance`

Suggested additional tags when useful:

- `#skill/chords`
- `#skill/rhythm`
- `#skill/scales`
- `#skill/fingerstyle`
- `#skill/repertoire`
- `#skill/fretboard`
