# Numeric Response Format

Use numbered multiple-choice questions during live practice so the user can reply quickly with digits only.

## Standard answer pattern

- Keep each question to 3 to 6 options.
- Number every option.
- Ask for one compact reply line, such as: `2, 4, 1, 3`
- Keep the option order stable when possible so the user learns the codes.

## Preferred scales

### Difficulty
1. Very easy
2. Manageable
3. Challenging but workable
4. Too hard

### Accuracy / cleanliness
1. Clean most of the time
2. A few mistakes
3. Frequent mistakes
4. Fell apart

### Timing / tempo control
1. Steady
2. A little rushed or dragged
3. Unstable often
4. Lost the pulse

### Tension / discomfort
1. No tension
2. Mild tension
3. Noticeable tension
4. Pain - stop and reduce load

### Confidence / readiness
1. Not ready
2. Need more reps
3. Almost ready
4. Ready to advance
